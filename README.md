# [인프런] 리액트 챗봇 애플리케이션 만들기

## 개발 환경

- 리액트
- VSCODE

## 작동 기능

### 개인 정보를 입력하면 저장
- who are you? 또는 hello? 를 입력하면 해당 기능 사용가능 
- 이름, 이메일, 주소 입력
- 입력한 이름과 이메일을 언급하면서 대화 종료 

### 카드메시지로 링크 정보 제공 
- card message, card 를 입력하면 해당 기능 사용가능
- 유튜브 링크를 가진 3가지 카드 메시지 대답
